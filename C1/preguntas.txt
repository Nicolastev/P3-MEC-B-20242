## Completa el siguiente código para imprimir "Hola, Mundo" en la consola.

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println(<<Hola, Mundo>>);
    }
}

## Completa el siguiente código para sumar dos números enteros y mostrar el resultado en la consola.

public class Suma {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        int suma = <<a + b>>;
        System.out.println("La suma es: " + suma);
    }
}

## Completa el siguiente código para definir un método estático que calcule el cuadrado de un número entero.

public class Calculadora {
    public static int cuadrado(int numero) {
        return <<numero * numero>>;
    }

    public static void main(String[] args) {
        int resultado = cuadrado(4);
        System.out.println("El cuadrado es: " + resultado);
    }
}
